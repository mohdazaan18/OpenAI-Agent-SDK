1. Hosted MCP

Client -> OpenAI -> MCP server -> MCP response -> OpenAI -> Client



2. Streamable HTTP MCP servers 

Client -> OpenAI server (MCP tool request) -> Client -> Will call MCP server endpoint
-> MCP Response -> Send back to OpenAI Server -> Client



3. Stdio MCP servers

Client -> Spawns MCP server as local process -> Client sends request via Stdio (stdin) -> MCP server processes request -> MCP server sends response via Stdio (stdout) -> Client
